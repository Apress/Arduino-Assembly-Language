A sketch to toggle an LED using a single wire
as a capacitance switch.

Based on Nerd Ralph's code at:
https:github.com/nerdralph/nerdralph/blob/master/avr/touch.c

See also this blog article on AVR parasitic capacitance:
https:nerdralph.blogspot.com/2015/10/parasitic-capacitance-of-avr-mcu-pins.html
