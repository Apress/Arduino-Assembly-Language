An example of Assembly code which returns a longish result of
more than eight bytes from a function, this demonstrates how
to return parameters in memory as opposed to in registers.

This version passes one parameter to show that the pointer to be
returned is always in the same registers---R24:R25.

Various disassembly files are provided to show how the 
Assembly code returns values.
