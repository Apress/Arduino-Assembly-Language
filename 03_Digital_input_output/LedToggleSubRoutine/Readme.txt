LedToggleSubRoutine.ino

This sketch shows how to toggle an LED when a switch, attached to 
D8/PB0 is pressed. The switch is pulled down using an external 
10K resistor. The switch is debounced using a software delay, 
which is written as a separate subroutine. 

This shows how to call subroutines in Assembly code. Uses the 
LED on D13/PB5 for the demo.
